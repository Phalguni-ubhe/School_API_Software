Instructions

File Requirements

  • Only PDF files are accepted
  • Each file should contain results for a single class
  • Make sure the PDF is clear and readable

Upload Process

  1. Select the Academic Year from the dropdown
  2. Click "Choose PDF" for the respective class
  3. Verify the selected file name
  4. Click "Upload Results" to submit
  5. Wait for the success message

Important Notes

  • Uploading a new file will replace any existing file for that class
  • The system will automatically process the results after upload
  • You can view the processed results in the dashboard
  • Make sure to check the file list below for existing uploads

Processing Information

After upload, the system will:

  • Extract student information and marks
  • Calculate subject-wise API
  • Generate performance reports
  • Update the dashboard statistics
